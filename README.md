# Anti Misinformation Bots
A group of different social media bots that assist in combating misinformation.

# Bots
## Discord
* Manual
* Slash and prefix commands
* Fights misinformation
* Trolls people spreading misinformation

## Reddit
* Automated
* Fights misinformation
